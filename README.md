# Neon Trench Defense

This repository contains the web version of the Neon Trench Defense game.

Live demo: https://qh48cdbvfm-hub.github.io/Neon-Trench-Defense/

The live demo is deployed with GitHub Pages.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Preview Production Build

1. Build the app:
   `npm run build`
2. Preview the build locally:
   `npm run preview`

## Deployment

The project is published to the `gh-pages` branch on GitHub, which serves the game at the repository GitHub Pages URL.
